# North Star

Build a single-page Agent Status Dashboard. A standalone HTML page that reads a
skill-graph.json file and displays an agent profile card with level, XP progress
bar, class, rarity-colored skill list, and a mini force-directed graph. It should
work by double-clicking the HTML file in Finder (no server needed). Dark theme,
bioluminescent aesthetic matching the AI Skill Forge visualization. The page
should be self-contained (inline CSS, inline JS, D3 from CDN).
